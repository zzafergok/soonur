# Soonur 🚀

> **"Ne kadar kaldı?"** sorusunun en şık cevabı.

Soonur, Türkiye'deki önemli sınavları (KPSS, ALES, YKS vb.), resmi tatilleri ve özel günleri takip etmenizi sağlayan, modern teknolojilerle "gelecekten" gelmiş bir geri sayım uygulamasıdır.

## 🎯 Nedir Bu?

Sürekli takvime bakıp "Sınava kaç gün kaldı?", "Bayram tatili ne zaman?" diye hesap yapmaktan sıkıldıysanız, Soonur sizin için burada. Sadece gün saymakla kalmaz, size görsel bir şölen sunar.

## ✨ Özellikler

- **Sınav Takibi:** ÖSYM'nin karmaşık takvimini, sizin için sade ve anlaşılır kartlara dönüştürdük. (KPSS, YDS, ALES ve daha fazlası).
- **Akıllı Gruplama:** Sınavlar kategorilerine göre (Ön Lisans, Ortaöğretim, Lisans) akıllıca gruplanır.
- **Hero Slider:** En önemli etkinlikleri gözünüzün önünden ayırmayın.
- **Modern UI:** Göz yormayan, ferah ve estetik tasarım.
- **Detay Sayfaları:** Her etkinlik için özel, saniyesine kadar takip edebileceğiniz detay sayfaları.

## 🛠 Teknoloji Yığını (Bleeding Edge Experience)

Bu proje sadece bir takvim değil, aynı zamanda bir teknoloji gösterisidir. Frontend dünyasının en güncel (hatta beta/alpha) araçlarını kullanıyoruz:

- **Framework:** [Next.js 16](https://nextjs.org/) (Evet, yanlış duymadınız. Gelecek burada.)
- **UI Library:** [React 19](https://react.dev/) (Hooks, Server Components ve daha fazlası.)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (Daha hızlı, daha hafif.)
- **Components:** [Radix UI](https://www.radix-ui.com/) (Erişilebilirlik bizim işimiz.)
- **Icons:** [Lucide React](https://lucide.dev/) (Piksel mükemmelliği.)
- **State:** [Zustand](https://docs.pmnd.rs/zustand) (Redux boilerplate'ine son.)
- **Validation:** [Zod](https://zod.dev/) & [React Hook Form](https://react-hook-form.com/) (Formlar güvende.)

## 🚀 Kurulum

Projeyi yerel makinenizde çalıştırmak için terminalinizi hazırlayın:

1.  **Repoyu klonlayın:**

    ```bash
    git clone https://github.com/zzafergok/soonur.git
    cd soonur
    ```

2.  **Paketleri yükleyin:**

    ```bash
    npm install
    # veya
    bun install
    # veya
    pnpm install
    # veya
    yarn
    ```

3.  **Geliştirme sunucusunu başlatın:**

    ```bash
    npm run dev
    ```

4.  Tarayıcınızda [`http://localhost:3000`](http://localhost:3000) adresine gidin ve büyüyü izleyin! ✨
